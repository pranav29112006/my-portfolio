# Pranav Titre — Portfolio

Personal portfolio website built with HTML, CSS, Tailwind CSS, and vanilla JavaScript.

## Features

- Hero section with profile photo
- About section with skills overview
- Projects showcase
- Contact form
- Responsive design
- Modular section files in `sections/`

## Getting Started

```bash
npm install
npm start
```

Then open [http://localhost:3000](http://localhost:3000) in your browser.

## Project Structure

```
Portfolio/
├── assets/
│   └── profile.png
├── sections/
│   ├── hero.html
│   ├── about.html
│   ├── projects.html
│   └── contact.html
├── index.html
├── script.js
├── styles.css
├── tailwind.config.js
└── package.json
```

## Customization

Edit the section files in `sections/` to update your bio, projects, and contact details.
